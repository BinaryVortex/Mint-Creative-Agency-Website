# Mint Creative Agency Website

A modern, responsive website for Mint Creative Agency, showcasing creative services, portfolio, and contact information. Built with React and Vite for fast performance and a sleek user experience.

## 🚀 Features

- **Responsive Design**: Optimized for all devices, from mobile to desktop.
- **Modern UI Components**: Utilizes Material-UI, Radix UI, and custom Tailwind CSS styling for a professional look.
- **Interactive Sections**:
  - Hero section with engaging call-to-action.
  - Services overview highlighting agency offerings.
  - Portfolio/Work showcase.
  - About section detailing the agency's story.
  - Contact form for client inquiries.
  - Footer with additional links and information.
- **Fast Build Tool**: Powered by Vite for quick development and production builds.
- **Component Library**: Extensive use of shadcn/ui components for consistency.

## 🛠️ Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS, Material-UI (@mui/material), Emotion
- **UI Components**: Radix UI primitives, shadcn/ui
- **Icons**: Lucide React, Material-UI Icons
- **Animations**: Motion library for smooth transitions
- **Forms**: React Hook Form with validation
- **Other Libraries**: Recharts for data visualization, React Slick for carousels, and more

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/BinaryVortex/Mint-Creative-Agency-Website.git
   cd Mint-Creative-Agency-Website
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   pnpm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser to view the website.

## 🏗️ Build for Production

To build the app for production:

```bash
npm run build
# or
pnpm build
```

The built files will be in the `dist` directory.
